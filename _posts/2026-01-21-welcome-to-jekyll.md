This is a sample blog post. You can create new blog posts by adding files to the `_posts` directory following the naming convention: `YYYY-MM-DD-title.md`

## Using Jekyll Templates

With Jekyll, you can now:
- Create reusable layouts and includes
- Use variables from `_config.yml`
- Write blog posts in Markdown
- Use Liquid templating for dynamic content

## Getting Started

1. Edit `_config.yml` to customize your site settings
2. Modify layouts in `_layouts/` directory
3. Add reusable components in `_includes/`
4. Write blog posts in `_posts/` following the naming convention

Happy building!
